# NordicBlocks Public Docs Index

This public repository keeps only the minimum product-facing documentation needed for installation and release operations.

## Included

1. install docs in docs/install/;
2. release docs in docs/releases/.

## Excluded by policy

This public repository intentionally excludes:

1. internal worklogs;
2. roadmap drafts and planning notes;
3. family-level exploration documents;
4. server rollout notes;
5. temporary migration and scratch files.
